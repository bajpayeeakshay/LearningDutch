## 3.1 Days of the Week (De dagen van de week)

⚠️ **NEVER capitalize** days in Dutch (unless starting a sentence)!  
Use preposition **"op"** for days

| English   | Dutch     | Pronunciation | Memory Trick           |
|:----------|:----------|:--------------|:-----------------------|
| Monday    | maandag   | MAAN-dahkh    | maan = moon (Moon-day) |
| Tuesday   | dinsdag   | DINS-dahkh    |                        |
| Wednesday | woensdag  | VOONS-dahkh   | ⚠️ W = V sound!        |
| Thursday  | donderdag | DON-der-dahkh | donder = thunder       |
| Friday    | vrijdag   | FRAY-dahkh    | vrij = free            |
| Saturday  | zaterdag  | ZAH-ter-dahkh |                        |
| Sunday    | zondag    | ZON-dahkh     | zon = sun              |

**Usage:** "op maandag" (on Monday), "elke dinsdag" (every
Tuesday)\[^33\]\[^30\]

## 3.2 Months of the Year (De maanden van het jaar)

⚠️ **NEVER capitalize** months in Dutch!  
Use preposition **"in"** for months

| English   | Dutch     | Pronunciation  |
|:----------|:----------|:---------------|
| January   | januari   | YAH-nu-ah-ree  |
| February  | februari  | FAY-bru-ah-ree |
| March     | maart     | MAART          |
| April     | april     | ah-PRIL        |
| May       | mei       | MAY            |
| June      | juni      | YU-nee         |
| July      | juli      | YU-lee         |
| August    | augustus  | ow-KHUHS-tuhs  |
| September | september | sep-TEM-ber    |
| October   | oktober   | ok-TOH-ber     |
| November  | november  | noh-VEM-ber    |
| December  | december  | day-SEM-ber    |

**Usage:** "in januari" (in January), "in de zomer" (in
summer)

## 3.4 Parts of the Day & Time Expressions

| Dutch                  | English            | Pronunciation       |
|:-----------------------|:-------------------|:--------------------|
| de ochtend / de morgen | morning            | OKHT-ent / MOR-khen |
| de middag              | noon/afternoon     | MID-dahkh           |
| de namiddag            | afternoon          | NAH-mid-dahkh       |
| de avond               | evening            | AH-vont             |
| de nacht               | night              | nahkht              |
| middernacht            | midnight           | MID-der-nahkht      |
| vandaag                | today              | vahn-DAAKH          |
| morgen                 | tomorrow           | MOR-khen            |
| gisteren               | yesterday          | KHIS-ter-en         |
| overmorgen             | day after tomorrow | OH-ver-mor-khen     |
| nu                     | now                | nuw                 |
| later                  | later              | LAH-ter             |
| straks                 | soon/later         | strahks             |
| altijd                 | always             | AHL-tayt            |
| nooit                  | never              | noyt                |
| soms                   | sometimes          | soms                |
| vaak                   | often              | faak                |
| elke dag               | every day          | EL-kuh dahkh        |
| per week               | per week           | per vayk            |

**Telling Time:**

-   Hoe laat is het? = What time is it?
-   Het is drie uur = It is 3 o'clock
-   Het is half vier = It is 3:30 (⚠️ "half four" = HALFWAY TO four!)
-   Om 19.00 uur = At 7 PM (Write with period, NOT colon!)\[^18\]
