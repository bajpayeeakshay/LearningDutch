## 4.1 Greetings (Begroetingen)

| Dutch       | English        | Pronunciation      | When to Use      |
|:------------|:---------------|:-------------------|:-----------------|
| Hallo       | Hello          | hah-LOH            | Neutral — anyone |
| Hoi         | Hi             | hoy                | Casual — friends |
| Dag         | Hello/Goodbye  | dahkh              | Universal        |
| Goedemorgen | Good morning   | KHOO-duh-mor-khen  | Before noon      |
| Goedemiddag | Good afternoon | KHOO-duh-mid-dahkh | 12–6 PM          |
| Goedenavond | Good evening   | KHOO-dun-AH-vont   | After 6 PM       |
| Welkom      | Welcome        | VEL-kom            | Hosting          |

## 4.4 Introducing Yourself (Speaking Exam Essential!)

| Dutch                                      | English                         |
|:-------------------------------------------|:--------------------------------|
| Hoe heet je? / Hoe heet u?                 | What is your name?              |
| Ik heet \[naam\] / Mijn naam is \[naam\]   | My name is \[name\]             |
| Waar kom je vandaan?                       | Where do you come from?         |
| Ik kom uit India                           | I come from India               |
| Waar woon je?                              | Where do you live?              |
| Ik woon in Bleiswijk / Den Haag            | I live in Bleiswijk / The Hague |
| Wat doe je? / Wat is je beroep?            | What do you do?                 |
| Ik werk als data engineer                  | I work as a data engineer       |
| Hoe oud ben je?                            | How old are you?                |
| Ik ben \[^35\] jaar (oud)                  | I am \[^35\] years old          |
| Ben je getrouwd?                           | Are you married?                |
| Ja, ik ben getrouwd / Nee, ik ben ongehuwd | Yes / No                        |
| Heb je kinderen?                           | Do you have children?           |
| Spreek je Nederlands?                      | Do you speak Dutch?             |
| Ik leer Nederlands                         | I am learning Dutch             |
| Ik spreek een beetje Nederlands            | I speak a little Dutch          |
| Aangenaam (kennis te maken)                | Nice to meet you                |

\[^47\]\[^52\]

## 4.5 Useful Daily Phrases

| Dutch                      | English                       |
|:---------------------------|:------------------------------|
| Hoe gaat het (met je/u)?   | How are you?                  |
| Goed, dank je. En met jou? | Good, thanks. And you?        |
| Het gaat goed/prima/lekker | It's going well               |
| Niet zo goed               | Not so well                   |
| Ik ben moe                 | I am tired                    |
| Ik begrijp het             | I understand                  |
| Ik begrijp het niet        | I don't understand            |
| Wat bedoel je?             | What do you mean?             |
| Kun je langzamer praten?   | Can you speak slower?         |
| Kunt u dat herhalen?       | Can you repeat that?          |
| Ik weet het niet           | I don't know                  |
| Spreekt u Engels?          | Do you speak English?         |
| Waar is...?                | Where is...?                  |
| Hoeveel kost het?          | How much does it cost?        |
| Links / Rechts / Rechtdoor | Left / Right / Straight ahead |
| Laat maar                  | Never mind                    |
| Rustig maar                | Take it easy                  |
| Even kijken                | Let me see                    |
